# Trading Bot Algorítmico Conservador

Proyecto educativo de trading algorítmico orientado a gestión de riesgo y capital reducido.

## Características
- Estrategia conservadora (ETFs)
- Gestión de riesgo por operación
- Journal automático
- Dashboard web
- Paper trading

## Requisitos
- Python 3.10+
- Interactive Brokers (Paper)
